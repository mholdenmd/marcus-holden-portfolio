Overview

Project Overview

This project involved designing a clean, accessible mobile app interface focused on reducing cognitive load and improving task completion rates. The goal was to create an intuitive experience that guides users naturally from one action to the next without friction.

Starting from wireframes and building up to high-fidelity mockups, every design decision was grounded in UX principles — clear hierarchy, consistent patterns, and purposeful use of color and typography.

Challenge & Response
The Problem

What needed solving?

Mobile users are easily overwhelmed by cluttered interfaces and unclear navigation. Too many apps prioritize features over usability, leaving users frustrated and disengaged.

The Solution

How I solved it

Designed a minimal, focused interface that prioritizes the most important user actions. Clear visual hierarchy, intuitive navigation patterns, and consistent components make the experience feel effortless.

Reflection

What I Learned

01

Wireframes Save Time

Starting with low-fidelity wireframes prevented costly design decisions later and kept the focus on structure before style.

02

Less Is More

Removing unnecessary elements made the interface cleaner and more usable — every screen element needs to earn its place.

03

Design for the User

Every decision — from font size to button placement — should be driven by what makes the experience easier for the person using it.
